# 001 カウンター（+ / − / リセット）

## 概要

React の基本である **状態管理（state）** を理解するためのシンプルなカウンターアプリです。

ボタン操作によって数値を増減・リセットできるようにし、React コンポーネントの基本的な構造を学ぶことを目的としています。

---

## 問題

カウンターアプリ を作成してください。

---

## 目的

- React の `useState` を正しく使えるようになる
- 状態変更による再レンダリングの流れを理解する
- コンポーネントの基本構造に慣れる
- 「UI とロジックを分けて考える」感覚を身につける

---

## 達成条件

- 「+」ボタンでカウントが増える
- 「−」ボタンでカウントが減る
- 「リセット」ボタンで 0 に戻る
- 状態は React の `useState` で管理されている
- 画面がリロードされてもエラーなく動作する

---

## 使用技術

- React
- TypeScript
- Vite
- Tailwind CSS

---

## 機能

- カウントのインクリメント（+1）
- カウントのデクリメント（-1）
- カウントのリセット
- 現在の数値をリアルタイム表示
- シンプルで見やすい UI
