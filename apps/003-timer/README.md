# 003 タイマー（開始・停止・リセット）

## 概要

開始・停止・リセットができるシンプルなタイマーアプリです。

時間の経過を state として管理し、`useEffect` を使った副作用（タイマー処理）の扱いに慣れることを目的とします。

クリック連打ゲームで学んだタイマー制御を、

**より単純な要件に落とし込んで理解を深める** 位置づけのアプリです。

---

## 問題

タイマー（開始・停止・リセット）を作成してください。

---

## 目的

- `useState` を使った時間の状態管理に慣れる
- `useEffect` を使ったタイマー処理の基本を理解する
- 「開始 / 停止 / リセット」という状態遷移を整理して実装する
- 副作用（setTimeout / setInterval）のクリーンアップを意識する

---

## 達成条件

- 「開始」ボタンでタイマーが動き出す
- 「停止」ボタンでタイマーが止まる
- 「リセット」ボタンで時間が初期値に戻る
- タイマーは 1 秒ごとに正しく更新される
- 停止中は時間が進まない
- リロードしてもエラーなく動作する

---

## 使用技術

- React
- TypeScript
- Vite
- Tailwind CSS

---

## 機能

- タイマー開始
- タイマー停止
- タイマーリセット
- 経過時間の表示（秒）
- 現在の状態（動作中 / 停止中）が分かる UI
