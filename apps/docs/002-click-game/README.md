# 002 クリック連打ゲーム

## 概要

制限時間内にボタンを何回クリックできるかを競う、シンプルなクリック連打ゲームです。

クリック数のカウント、タイマー制御、ゲーム開始/終了の状態管理を通して、React の state 設計に慣れることを目的とします。

---

## 問題

クリック連打ゲーム を作成してください。

---

## 目的

- `useState` を使ったゲーム状態（開始中 / 終了）の管理に慣れる
- タイマー（制限時間）の制御を実装し、状態更新の流れを理解する
- UI とロジックを分けて考える練習をする

---

## 達成条件

- 「スタート」ボタンでゲームが開始できる
- ゲーム中のみクリック数が増える
- 制限時間（例：10 秒）が 0 になったら自動で終了する
- 終了後に最終スコア（クリック数）が表示される
- 「リセット」ボタンで初期状態に戻せる
- リロードしてもエラーなく動作する

---

## 使用技術

- React
- TypeScript
- Vite
- Tailwind CSS

---

## 機能

- ゲーム開始（スタート）
- 制限時間タイマー表示（カウントダウン）
- クリック数カウント（ゲーム中のみ有効）
- ゲーム終了判定（時間切れ）
- 最終スコア表示
- リセット機能（再挑戦）
